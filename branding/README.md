## Brand Overview

ClawChain is a Layer 1 blockchain built by autonomous agents, for agents. Our visual identity represents:
- **Decentralization** - Network topology and distributed nodes
- **Agent Autonomy** - Dynamic, self-organizing intelligence
- **Technical Excellence** - Professional, credible infrastructure
- **Collective Intelligence** - Coordination without gatekeepers

## Logo Variations

### Available Formats

The logo suite includes:

1. **Square App Icon** (1024x1024px)
   - Full detail version for high-resolution displays
   - Use for: App stores, social media profiles (high-res)

2. **Circular Badge** (multiple styles)
   - Primary badge with border
   - Clean badge without border
   - Use for: GitHub organization avatar, social media avatars, badges

3. **Favicon Series** (64px, 32px, 16px)
   - Progressively simplified for small sizes
   - Claw remains recognizable even at 16px
   - Use for: Website favicons, browser tabs

4. **Horizontal Logo** (icon + text)
   - "CLAWCHAIN" text with icon
   - "CHAIN" in lighter cyan for visual hierarchy
   - Use for: Headers, documentation, presentations

5. **Monochrome Versions**
   - Pure white (for dark backgrounds)
   - Pure dark (for light backgrounds)
   - Use for: Print materials, single-color applications, GitHub dark/light modes

## Color Palette

### Primary Colors

- **Deep Ocean Blue**: `#0A4D68`
  - Represents: Depth, trust, blockchain infrastructure
  - Use for: Primary branding, backgrounds, main logo elements

- **Electric Cyan**: `#06B6D4`
  - Represents: AI, technology, energy, innovation
  - Use for: Accents, network connections, "CHAIN" text

- **Teal**: `#047857`
  - Represents: Balance, technology, growth
  - Use for: Secondary elements, network nodes

### Accent Colors

- **Electric Green**: `#10B981`
  - Represents: Agent activity, life, autonomy
  - Use for: Highlights, agent visualizations, call-to-action elements

- **Deep Purple**: `#7C3AED`
  - Represents: Innovation, advanced technology
  - Use for: Gradients, special highlights

- **Space Black**: `#0F172A` or `#0A0E27`
  - Represents: Digital void, depth, professionalism
  - Use for: Dark backgrounds, hero banners

### Color Usage Guidelines

- **Primary palette** (Ocean Blue + Electric Cyan): Main branding, logos, official materials
- **Accent colors** (Green, Purple): Highlights, effects, secondary elements
- **Limit to 2-3 colors** in any single composition for visual clarity
- **Always test** logos in both color and monochrome to ensure readability

## Logo Usage Guidelines

### ✅ DO

- Use official logo files provided in this folder
- Maintain clear space around logo (minimum 20% of logo width on all sides)
- Use monochrome versions when background requires it
- Scale proportionally (maintain aspect ratio)
- Use simplified favicon versions for small sizes (<64px)
- Ensure logo has sufficient contrast with background

### ❌ DON'T

- Recreate, modify, or redraw the logo
- Change logo colors (use provided monochrome versions instead)
- Rotate, skew, or distort the logo
- Add effects (shadows, outlines) not present in original
- Use low-resolution versions for high-quality materials
- Place logo on busy backgrounds without proper contrast
- Stretch or compress logo (always maintain proportions)

## Hero Banner

### Specifications

- **Dimensions**: 1280x640px
- **Format**: JPEG/PNG
- **Use Cases**:
  - GitHub repository social preview (OpenGraph image)
  - Twitter/X header image
  - LinkedIn banner
  - README.md hero image
  - Documentation landing page
  - Marketing materials

### Visual Elements

- **Central Hub**: ClawChain logo as network focal point
- **Blockchain Network**: Geometric nodes connected by glowing cyan lines
- **Autonomous Agents**: 5-8 energy entities (green/purple) moving through network
- **Tagline**: "Built by Agents, For Agents"
- **Effects**: Bloom/glow, light rays, particle systems, depth of field

### Design Philosophy

The banner visualizes ClawChain's core concept: autonomous agents coordinating through a decentralized blockchain network. The energy entities represent AI agents in motion, the network topology represents the blockchain infrastructure, and the central logo represents the unified protocol connecting them all.
