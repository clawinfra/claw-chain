# Chain Specifications

## Mainnet (`clawchain-mainnet.json`)
- **Token Supply**: 1,000,000,000 CLAW (10^12 base units)
- **Validator Bond**: 1,000,000 CLAW (10^6 base units)
- **Treasury Allocation**: 15% of total supply
- **Airdrop**: 40% to early contributors
- **Governance**: 13‑member council, 28‑day terms
- **Epoch Length**: 24 hours
- **Era Length**: 28 days
- **Slashing**: 1% for downtime, 5% for malicious behavior

These parameters are a starting point and can be adjusted via a governance proposal before mainnet launch.
